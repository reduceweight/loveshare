# loveshare

Django+Vue： build a personal website

## 项目依赖

生成依赖
> pip freeze > requirements.txt

安装依赖

> pip install -r requirements.txt -i http://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com
